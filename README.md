# Mess Management System

A scalable, production-ready Mess Management Mobile Application for managing 500+ students with subscription-based coupon system.

## Tech Stack

- **Frontend**: React Native
- **Backend**: Node.js + Express.js
- **Database**: MongoDB
- **Cache**: Redis
- **Additional**: Email Notifications, QR Code System, Payment Gateway
